What is ZipLog ?
It is a command line utility which will help you to zip and tar your aged logs which is consuming your Windows FileSystem.

Do I need any specific license to use this ?
No! The tool is completely free to use and you can even download the source code and modify it as per your use.

Will I have to be familiar with the commands and syntaxes for any of the activity related to this in order to use ZipLog ?
No. You'll be guided throughout. Don't worry. You'll be fine :)


---------------------------------------------------------------------------
---------------------------------------------------------------------------

How to use this tool ?
- Place the entire directory in your Windows machine (location doesn't really matter).
- Open the ZipLog/Windows directory.
- Run the script called ziplog.ps1 using Powershell with required privileges.
- One you run the above script in brackets, you'll be guided throughout. It'll ask you the aboslute path (i.e. complete path starting from your root filesystem) and then it'll ask you to enter the no. of days before which you don't want any log files and want to ZipLog them in the form of .zip

Voila ! Simple right ? :)
And that's how ZipLog works !
Enjoy :)
